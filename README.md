# elementor

AutoPOM-Agent implementation workspace.

## Quick Start

```bash
PYTHONPATH=src python3 -m autopom.cli.main --base-url "https://example.com" --output-dir "output"
```

This will generate:

- JSON page models in `output/models_json`
- Java POM classes in `output/java`
- crawl report in `output/reports/crawl_summary.md`
